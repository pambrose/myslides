🔦 What is Redis?

  • REmote DIctionary Server
  • Fast NoSQL database written in ANSI C
  • An in-memory database -- all data is stored in RAM
  • Based on the key-value model
  • All access is via a key
  • Meant to be run as a network-atached server
  • Simple TCP protocol, which is implmented in many languages
  • Can act as a cache, database, pub-sub, and message broker

SET/GET Commands (1) 


redis> SET key1 "Hello"
OK
redis> SET key2 "World"
OK
redis> GET key1
"Hello"
redis> GET key2
"World"
redis> GET nonexisting
(nil)
redis>

SET/GET Commands (2) 


redis> SET tempkey "Temp val with 10 secs of life" EX 10
OK
redis> SET cond1key "Set only if key does not exist" EX 10 NX
OK
redis> SET cond2key "Set only if key does exist" EX 10 XX
OK
redis>

SET/GET Commands (2) 


redis> MSET key1 "Hello" key2 "World"
OK
redis> MGET key1 key2 nonexisting
1) "Hello"
2) "World"
3) (nil)
